## ğŸ“‹ How to Print Documents

### Method 1: Copy-Paste to Word
1. Open the .md file in any text editor
2. Select all (Ctrl+A / Cmd+A)
3. Copy (Ctrl+C / Cmd+C)
4. Open Microsoft Word
5. Paste (Ctrl+V / Cmd+V)
6. Format headers using Word Styles
7. Print

### Method 2: Use Markdown Viewer
1. Install a Markdown viewer (e.g., Markdown Preview Plus)
2. Open the .md file
3. Export to PDF or print directly

### Method 3: Online Converter
1. Use an online Markdown to Word converter
2. Upload the .md file

- Bullet lists paste automatically
- Numbered lists paste automatically
- Adjust spacing as needed

### Tables
- Markdown tables may need manual formatting
- Use Word's "Convert Text to Table" feature
- Adjust column widths

### Links
- Markdown links: `[text](url)`
- In Word: Keep as-is or convert to hyperlinks
