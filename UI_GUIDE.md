## Quick Start

### 1. Start the API Server

```bash
uvicorn src.api.app:app --host 127.0.0.1 --port 8001 --reload
```

### 2. Open in Browser

Simply navigate to:
```
http://localhost:8001/
```

The UI will automatically load! No need to go to `/docs` - the root URL now serves the beautiful dashboard.

## Usage Tips

1. **Refresh Data**: Click the "Refresh" button on any tab to reload data
2. **Filter Results**: Use the filter controls to narrow down results
3. **View Details**: Click on any row to see more details (future enhancement)
4. **Monitor Status**: Check the status bar at the top for API health

## Troubleshooting

### UI Not Loading
- Make sure the server is running
- Check browser console for errors
- Try hard refresh (Ctrl+F5 or Cmd+Shift+R)

### No Data Showing
- This is normal if the database is empty
- Run model training to generate predictions
- Execute trades to see trade history

### API Errors
- Check that greenlet is installed: `pip install greenlet`
- Verify database connection (optional - API works without DB)
- Check server logs for detailed error messages
