## üìù Document Formatting Notes

All documents are formatted in Markdown (.md) format. To print in Word:

1. **Open the .md file** in a text editor or Markdown viewer
2. **Copy all content** (Ctrl+A, Ctrl+C)
3. **Paste into Word** (Ctrl+V)
4. **Format as needed**:
   - Headers will paste as bold text
   - Code blocks may need manual formatting
   - Lists will paste as bullet points
   - Tables may need adjustment

### Word Formatting Tips:
- Use Word's "Styles" to format headers (Heading 1, Heading 2, etc.)
- Code blocks can be formatted with "Courier New" font
- Tables can be converted using Word's "Convert Text to Table" feature
- Use "Find & Replace" to format markdown links if needed
